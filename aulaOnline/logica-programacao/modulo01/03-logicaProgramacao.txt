2. Elaborada a declaração das variáveis;
3. Utilizado o comando de atribuição.

nome (variável para armazenar o nome completo)
cpf (variável para armazenar o CPF)
rg (variável para o RG)
altura (variável para a altura)
endereco (variável para armazenar o endereço completo)
